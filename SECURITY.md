# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in motel, please report it
responsibly. **Do not open a public issue.**

Please use [GitHub's private vulnerability reporting](https://github.com/andrewh/motel/security/advisories/new)
to submit a report. This keeps the details confidential until a fix is ready.

You should receive an acknowledgement within 48 hours.

## Supported Versions

Only the latest release is supported with security updates.
